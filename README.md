# 山东科技大学教务系统选课脚本

> 一个 Python 自动化选课工具，支持密码登录、课程查询、智能选课、自动抢课等功能。

## 功能特性

- **自动登录**：破解教务系统动态加密，支持密码登录和 Cookie 登录
- **配置文件**：自动保存账号密码到 `config.json`，下次启动免输入
- **选课轮次查询**：显示轮次名称、选课时间、退课权限、状态
- **多类型课程支持**：公选课、限选课/专业选修课、必修课、实践课
- **自动进入选课中心**：选择轮次后自动加载对应课程列表
- **智能选课**：
  - 关键词搜索（支持课程名/课头名/老师名/类别）
  - 自动检测时间冲突
  - 自动过滤人数已满课程
  - 自动选择正确的选课操作接口
- **智能自动抢课**：
  - 实时从页面提取细分类别（如人文、创新创业等）
  - 按 `szjylb` 参数精确查询某类别课程
  - 线上/线下课筛选
  - 一键选完或单选
- **学分学时显示**：课程列表和已选课程均显示学分和学时
- **退课功能**：支持查询已选课程并退课

## 功能详解

### 选项 4：进入选课中心

选择轮次序号后，自动进入选课中心并显示该轮次的课程列表。支持根据轮次名称自动判断课程类型：

| 轮次名称关键词 | 自动查询接口 |
|--------------|------------|
| 公选课 | 公选课 |
| 专业选修/外语/小语种/俄语/日语 | 限选课 |
| 体育/必修 | 必修课 + 实践课 |
| 实践/劳动 | 实践课 |

## 配置说明

### config.json

首次登录成功后选择保存，会在脚本同级目录生成 `config.json`：

```json
{
  "username": "202311100913",
  "password": "your_password"
}
```

**注意**：密码以明文形式存储在本地，请勿将配置文件上传到公共仓库。

## 技术实现

### 登录加密

教务系统使用动态加密参数（`scode` 和 `sxh`），脚本通过以下步骤实现自动登录：

1. 获取登录页面 HTML，提取 `scode` 和 `sxh`
2. Base64 编码用户名和密码
3. 按 `sxh` 指示的位置插入 `scode` 片段进行混淆加密
4. 提交加密后的数据完成登录

### 选课接口

不同课程类型使用不同的查询和操作接口：

| 课程类型 | 查询接口 | 操作接口 |
|---------|---------|---------|
| 公选课 | `xsxkGgxxkxk` | `ggxxkxkOper` |
| 限选课/专业选修 | `xsxkXxxk` | `xxxkOper` |
| 必修课 | `xsxkBxxk` | `bxxkOper` |
| 实践课 | `xsxkLdsjkxk` | `ldsjkOper` |

### 时间冲突检测

解析课程时间字符串（如 `"3-12周 星期六 9-10"`），提取周次、星期、节次信息，与已选课程逐一比对。
